# epic_free

## 用户说明
顾名思义，这个插件可以看 [EPIC Games](https://store.epicgames.com) 在送和将要送什么游戏，*喜加一什么的最爽了*。

## 搭建说明
虽然 epic_free 插件本身没有配置项，但你可以在 [daily](/docs/guide/plugins/daily#搭建说明) 插件的每日推送中加入 EPIC Games 喜加一的推送。
