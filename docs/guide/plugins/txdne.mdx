# txdne
This X Does Not Exist.

这个插件收录了一些 [This X Does Not Exist](https://thisxdoesnotexist.com/) 上的网站。

## 用户说明
目前支持 5 个网站，发送不带参数的 `/txdne` 查看支持的来源和它们的触发关键词。

:::info
和 [trending](/docs/guide/plugins/trending) 不一样，这个插件没法用中文原名触发，也就是说，`/txdne 猫` 是不行的，必须是 `/txdne cat` 或者 `/txdne c`。

~~也不知道是不是我不同时间写代码的脑回路不太一样。~~
:::

<Example>
  <Send>/txdne</Send>
  <Recv>
{`所有支持的来源：
猫 - cat、c
画作 - artwork、a
马 - horse、h
人 - person、p
房产 - rental、r`}
  </Recv>
</Example>

发送 `/txdne <来源>` 就可以得到随机一张由 AI 生成的不存在照片（或者其他什么东西），注意取决于不同的网站，这些图片可能不是实时生成而是提前准备的，所以**有可能**随机到相同的图片。

<Example>
  <Send>/txdne cat</Send>
  <Recv>

![txdne例图](/example/txdne.jpg)

  </Recv>
</Example>
