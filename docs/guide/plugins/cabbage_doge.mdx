# cabbage_doge
菜狗，带RGB的

## 用户说明

<Command name="菜狗" brief="生成彩色菜狗GIF" perm="cabbage_doge">

/菜狗 - 生成随机颜色的菜狗
/菜狗 &lt;颜色&gt; - 生成指定颜色的菜狗
/菜狗 &lt;多个颜色&gt; - 生成渐变色的菜狗
颜色可以是16进制，也可以是CSS颜色

</Command>

颜色可以使用：
* 3 位 16 进制：`#123`（等价于 `#112233`）
* 6 位 16 进制：`#123456`
* CSS 颜色名（[列表](https://developer.mozilla.org/en-US/docs/Web/CSS/named-color#value)）：`blue`
* rgb()：`rgb(12,34,56)`
* hsl()：`hsl(123deg,34%,56%)`
* Material Design 颜色名（[列表](https://mui.com/zh/material-ui/customization/color/#2014-material-design-color-palettes)）：`blue500`
  * 考虑到国内可能访问不了 [MD 官网](https://material.io/design/color/the-color-system.html)，上述链接是 Material-UI 的网站
