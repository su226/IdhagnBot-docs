### 关于诺基亚
由于 Nonebot2 会忽略命令名之后第一个字前的所有空格和空行，如果你需要诺基亚的开头是空格或空行，你可以在诺基亚后面加一个 `.` 点号。

<Example collapsed>
  <Send>/诺基亚.   ←这里有几个空格</Send>
  <Recv>

![](/example/meme_word/nokia_whitespace.png)

  </Recv>
</Example>

## 搭建说明
虽然 meme_word 本身没有配置文件，但 `/诺基亚` 需要一个叫 `pixel` 的[特殊字体](/docs/guide/misc#杂项方法)，推荐使用 [GNU Unifont](http://www.unifoundry.com/unifont/index.html)*（冷知识：Minecraft 的中文字体也是这个）*

`/osu` 可以设置叫 `osu` 的特殊字体，推荐设置为 Aller Bold（也就是 osu! 的 Logo 字体），如果没有设置，会 fallback 到 `sans bold`。

`/诺基亚` 依赖[结巴分词](https://github.com/fxsjy/jieba)库，使用 `pdm install -G jieba` 安装，未安装结巴分词会自动禁用这个命令。

`configs/misc.yaml`
```yaml
special_font:
  pixel: Unifont
  osu: Aller Bold
```
