# meme_text
文字梗图生成器

部分梗图来自 [nonebot-plugin-memes](https://github.com/noneplugin/nonebot-plugin-memes) 和 [nonebot-plugin-logo](https://github.com/noneplugin/nonebot-plugin-logo)，`/orly` 来自 [nanmu42/orly](https://github.com/nanmu42/orly)。

### 关于诺基亚
由于 Nonebot2 会忽略命令名之后第一个字前的所有空格和空行，如果你需要诺基亚的开头是空格或空行，你可以在诺基亚后面加一个 `.` 点号。

<Example collapsed>
<Send>/诺基亚.   ←这里有几个空格</Send>
<Recv>

![](/example/meme_text/nokia_whitespace.png)
</Recv>
</Example>

## 搭建说明
虽然 meme_text 本身没有配置文件，但 `/诺基亚` 需要一个叫 `pixel` 的[特殊字体](/docs/guide/misc#杂项方法)，推荐使用 [GNU Unifont](http://www.unifoundry.com/unifont/index.html)*（冷知识：Minecraft 的中文字体也是这个）*
