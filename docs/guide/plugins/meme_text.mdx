# meme_text
文字梗图生成器

## 用户说明
下文包含大量预览，全部均已折叠，点开时才会加载，流量党慎点。<br />
~~有钱到可疑的兽人控~~土豪、WiFi 党或电脑党可以点这里：

             [-color COLOR] [-image IMAGE] [-no-grayscale]
             title1 [title2]

positional arguments:
  title1                标题第一行
  title2                标题第二行

  -header HEADER        页眉

  -no-grayscale         不对下载的图片去色

</Command>

### 关于诺基亚
由于 Nonebot2 会忽略命令名之后第一个字前的所有空格和空行，如果你需要诺基亚的开头是空格或空行，你可以在诺基亚后面加一个 `.` 点号。

<Example input="诺基亚.   ←这里有几个空格" collapsed>

![](/img/example_meme_text/nokia_whitespace.png)

</Example>

## 搭建说明
虽然 meme_text 本身没有配置文件，但 `/诺基亚` 需要一个叫 `pixel` 的[特殊字体](/docs/guide/misc#杂项方法)，推荐使用 [GNU Unifont](http://www.unifoundry.com/unifont/index.html)（冷知识：Minecraft 的中文字体也是这个）
