# auto_repoke
自动回戳

## 用户说明
当有人戳一戳机器人时，机器人就会戳一戳他。

<Example>
  <Send><em>你戳了戳IdhagnBot的晶体管</em></Send>
  <Recv><em>IdhagnBot戳了戳你的猫猫头</em></Recv>
</Example>

:::note ~~老Bug~~ 冷知识
在很久很久之前，如果有人登录机器人的帐号并戳了戳机器人自己，机器人就会一直戳自己刷屏，直到机器人被关闭。

当然这个 Bug 早就被修复了。
:::
