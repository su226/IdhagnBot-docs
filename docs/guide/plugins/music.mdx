# music
点歌插件（目前只支持网易云）

## 用户说明
<Command name={["网易云", "netease", "163", "cloudmusic"]} brief="网易云点歌" perm="music.netease">{`\
/网易云 <关键字> - 搜索关键字
/网易云 <id> - 发送指定ID的歌
`}</Command>
