# meme_pic
头像梗图生成器

大部分梗图来自 [nonebot-plugin-petpet](https://github.com/noneplugin/nonebot-plugin-petpet)，`/卢浮宫`、`/做旧` 来自 [神奇海螺](https://lab.magiconch.com/)，还有少部分梗图为作者~~瞎~~整活。

## 用户说明
所有命令的帮助文档均位于子分类 `meme_pic`，可以使用 [`/help meme_pic`](/docs/guide/concepts/help) 查看，并且所有命令都使用类似于 UNIX Shell 的命令解析。支持**输入**动图的命令会在本页和 `/help` 中标注为 `[动]`（其余命令只会使用动图的第一帧）。**输出**动图的默认格式是 GIF，高级用户也可使用 APNG 或 WebP 格式，注意这两个格式在官方 QQ 客户端中可能不能正常显示，但在 [Icalingua++](https://github.com/Icalingua-plus-plus/Icalingua-plus-plus) 等第三方客户端是可以正常显示和保存的。

在指定用户时你可以使用 QQ 号、用户名、直接 @ 对方、嵌入图片或指定图片链接，另外还有这 4 个特殊值：
* `~`、`自己`、`我` 代表你自己（不指定**来源**时默认是这个）
* `0`、`机器人`、`bot`、`你` 代表机器人（不指定**目标**时默认是这个）
* `!`、`他`、`她`、`牠`、`祂` 代表回复的用户（你需要回复一条消息）
* `?`、`那个`、`它` 代表回复的内容（你需要回复一条有且只有一张图片的消息）
* `-`、`这个` 代表发送一张图片

部分命令需要一个名字，在使用 QQ 号或 @ 时会自动获取群名片或昵称，而嵌入图片、指定图片链接和发送图片时**必须**用参数指定名字。（通常叫 `--name`）有的命令还可选指定性别，来决定用 `他`、`她` 还是 `它`，同样会从资料中获取，也可以用参数手动指定，但并**不必要**，因为默认会使用 `它`。（通常叫 `--gender`，并且除了 `male`「他」、`female`「她」、`unknown`「它」，还可以用 `animal`「牠」和 `god`「祂」~~，我承认这两个有我的恶趣味~~）

## 搭建说明
`meme_pic` 并没有特别的配置，但以下命令需要 numpy 和 / 或 opencv，使用 `pdm install -G opencv` 来安装它们。
* `/腐蚀` 和 `/膨胀`
* `/鱼眼`
* `/卢浮宫`
* `/包浆`
* `/震惊`

你可能还需要查看 [用户别名](/docs/guide/misc#用户别名)、[字体渲染](/docs/guide/misc#字体渲染) 和 [图片采样](/docs/guide/misc#图片采样)
